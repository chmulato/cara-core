Como Montar um Dashboard de Vendas em Python para sua Loja

Cara Core Informática 82 seguidores
25 de agosto de 2025

Este guia mostra, de forma simples, como transformar suas planilhas de vendas em um painel visual usando Python. Mesmo que você não seja analista de dados, vai conseguir montar uma visão clara para tomar decisões melhores.

História rápida: uma pequena loja de produtos naturais tinha 8 planilhas mensais diferentes. O dono só sabia “se vendeu bem” olhando o saldo bancário no fim da semana. Depois de organizar e visualizar dados em um dashboard simples (ticket médio, produtos que mais giram e horas de pico), ele ajustou estoque e reduziu perda de itens perecíveis em 22% em 2 meses.

Objetivo do Dashboard

  • Responder rápido: “Como estão as vendas do mês vs. mês passado?”
  • Encontrar produtos campeões e encalhados.
  • Prever necessidade de reposição básica.
  • Monitorar ticket médio e número de pedidos.

Principais KPIs (indicadores)

  • Faturamento Total
  • Número de Pedidos
  • Ticket Médio (Faturamento ÷ Pedidos)
  • Top 5 Produtos (por receita ou quantidade)
  • Percentual por Categoria
  • Curva Horária ou Diário
  • Margem Bruta (%) (opcional se tiver custo)

Carregando e Preparando os Dados

import pandas as pd
from pathlib import Path

def carregar_vendas(arquivo):
    """Carrega dados de vendas e retorna DataFrame preparado."""
    df = (pd.read_excel(arquivo, parse_dates=["data"])
            .assign(
                receita=lambda x: x["quantidade"] * x["preco_unitario"],
                custo_total=lambda x: x["quantidade"] * x["custo_unitario"]
            )
            .assign(margem=lambda x: x["receita"] - x["custo_total"])
            .astype({
                "pedido_id": "str",
                "produto": "str", 
                "categoria": "str",
                "cliente_id": "str"
            }))
    return df

# Uso
df = carregar_vendas("vendas.xlsx")
print(f"Dados carregados: {len(df)} linhas, {df.columns.tolist()}")

Calculando Métricas

def gerar_resumo(df):
    """Pipeline de métricas usando pandas fluido."""
    return {
        "faturamento": df["receita"].sum(),
        "pedidos": df["pedido_id"].nunique(),
        "ticket_medio": df["receita"].sum() / df["pedido_id"].nunique(),
        "margem_pct": (df["margem"].sum() / df["receita"].sum()) * 100,
        "produtos_únicos": df["produto"].nunique(),
    }

def top_produtos(df, n=5):
    return (df.groupby("produto", as_index=False)["receita"]
              .sum()
              .sort_values("receita", ascending=False)
              .head(n))

def serie_temporal(df, freq="D"):
    return (df.groupby(df["data"].dt.date)["receita"]
              .sum()
              .reset_index())

# Executar pipeline
metricas = gerar_resumo(df)
top_5 = top_produtos(df)
vendas_diarias = serie_temporal(df)

print(f"Faturamento: R$ {metricas['faturamento']:,.2f}")
print(f"{metricas['pedidos']} pedidos, ticket médio R$ {metricas['ticket_medio']:.2f}")

Visualizando com Plotly

import plotly.express as px
import plotly.graph_objects as go
from plotly.subplots import make_subplots

def criar_dashboard(df):
    """Cria figuras do dashboard em uma função fluida."""
    
    # Dados para gráficos
    top_5 = top_produtos(df)
    cats = df.groupby("categoria")["receita"].sum().reset_index()
    temporal = serie_temporal(df)
    
    # Subplot com 3 gráficos
    fig = make_subplots(
        rows=2, cols=2,
        specs=[[{"colspan": 2}, None], 
               [{}, {}]],
        subplot_titles=("Receita Diária", "Top Produtos", "Por Categoria")
    )
    
    # Linha temporal
    fig.add_trace(
        go.Scatter(x=temporal["data"], y=temporal["receita"], 
                   mode="lines+markers", name="Receita"),
        row=1, col=1
    )
    
    # Barras top produtos
    fig.add_trace(
        go.Bar(x=top_5["produto"], y=top_5["receita"], name="Top 5"),
        row=2, col=1
    )
    
    # Pizza categorias
    fig.add_trace(
        go.Pie(labels=cats["categoria"], values=cats["receita"], name="Categorias"),
        row=2, col=2
    )
    
    fig.update_layout(height=600, showlegend=False, title="Dashboard de Vendas")
    return fig

# Gerar e exibir
dashboard = criar_dashboard(df)
dashboard.show()

Dashboard Simples em Página Única

Podemos juntar tudo em um único HTML exportado:

from pathlib import Path
from datetime import datetime

def exportar_dashboard(df, destino="dashboard_vendas.html"):
    """Pipeline completo: dados → dashboard → HTML."""
    
    # Gerar componentes
    metricas = gerar_resumo(df)
    fig = criar_dashboard(df)
    
    # Template com f-string
    template = f"""<!DOCTYPE html>
<html lang='pt-br'>
<head>
    <meta charset='utf-8'>
    <title>Dashboard Vendas - {datetime.now().strftime('%d/%m/%Y')}</title>
    <style>body{{font-family:Segoe UI,sans-serif;margin:20px;}}</style>
</head>
<body>
    <h1>Dashboard de Vendas</h1>
    <div style="display:flex;gap:20px;margin:20px 0;">
        <div>Faturamento: R$ {metricas['faturamento']:,.2f}</div>
        <div>Pedidos: {metricas['pedidos']}</div>
        <div>Ticket Médio: R$ {metricas['ticket_medio']:.2f}</div>
    </div>
    {fig.to_html(include_plotlyjs='cdn', div_id='dashboard')}
</body>
</html>"""
    
    Path(destino).write_text(template, encoding="utf-8")
    print(f"Dashboard salvo: {destino}")
    return destino

# Executar pipeline completo
arquivo = exportar_dashboard(df)

O resultado final é um dashboard interativo que transforma dados brutos em insights visuais claros. A função exportar_dashboard() automatiza todo o processo: carrega os dados, calcula métricas, gera gráficos e exporta um arquivo HTML pronto para uso.

Glossário Rápido

  • Dashboard: tela que resume informações em gráficos.
  • KPI: indicador-chave (número que mostra desempenho).
  • Ticket Médio: valor médio que cada pedido gera.
  • Categoria: grupo de produtos (ex.: bebidas, acessórios).
  • Margem Bruta: receita menos custo direto.

Resumo Final

Com poucas linhas de código você transforma dados soltos em visão clara. Comece pequeno (3 KPIs + 2 gráficos) e evolua. O mais importante é manter dados limpos e consistente atualização.
